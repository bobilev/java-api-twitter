Создавался как бото-вод акаунтов в твиттере
Есть эмуляция деятельности в Посты/фоловинг/анфоловинг (с учетом обхода ограничений API благодаря прокси которые программа сама парсила и проверяла)
программа работает по заданому расписанию,можно оставить и забыть
Есть мониторинг состояния аккаунтов
Цель программы прокачать твиттер акаунты, распростронение рекламыи слив траффика с твиттера для арбитража.
Программа работает в многопоточном режиме, потдерживала нормально около 450+ акков.
Все данные аккаунтов хранились в MySQL, остальные настройки сериализовывались
